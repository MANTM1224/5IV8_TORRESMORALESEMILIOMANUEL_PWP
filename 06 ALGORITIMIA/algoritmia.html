Peroblema 2

            

                Dados dos vectores v1=(x1, x2, x3, x4, ..., xN), v2=(y1, y2, y3, y4, ..., yN) y teniendo
                permitido permutar el orden de los elementos de dicho vector,
                encontrar el minimo producto escalar entre ambos verctores.
                

                el producto escalar se forma de la siguiente manera: x1y1 + x2y2 + x3y3 + ... + xNyN

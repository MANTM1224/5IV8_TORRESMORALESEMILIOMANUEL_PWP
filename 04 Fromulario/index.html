Todoslos formularios en la web, manejan metodos para el envio de las diferentes peticiones que realiza el usuario. Considerando la arquitectura cliente-servidor, el manejo de las peticiones se hace por medio del protocolo HTTP y actualmente por https, cada peticion debe ser atendida en un tiempo de respuesta conforme al modelo TCP/IP. los metodos del formulario SON

    • GET:

      este metodo se ecarga de enviar la informacion del formulario por medio de la salida esyandar (conocida como url), del navegador, esto propicia que ouedan existir, inyecciones de codigo realizando violaciones a la seguridad del sistema.

      POST:

      este metodo se encarga de enviar la informacion para poder registrar un cambio o un nuevo registro dentro del sistema, enviandola igual por el metodo estandar pero mediante un ocultamiento, esto significa que no es visible para el usuario. Lo que nos ofrece un mayor grado de seguridad en la informacion.
